# Data Directory

This directory contains stock market data files downloaded from Financial Modeling Prep API.

## Current Data Files

- `AAPL_daily_5y.csv` - Apple Inc. (5 years daily data)
- `AMZN_daily_5y.csv` - Amazon.com Inc. (5 years daily data)
- `GOOGL_daily_5y.csv` - Alphabet Inc. (5 years daily data)
- `MSFT_daily_5y.csv` - Microsoft Corporation (5 years daily data)
- `NVDA_daily_5y.csv` - NVIDIA Corporation (5 years daily data)

## File Naming Convention

`{SYMBOL}_daily_{N}y.csv` - Daily OHLCV data for N years

## CSV Format

Files are formatted for NeuralProphet with the following columns:
- `ds` - Date in YYYY-MM-DD format (timestamp)
- `y` - Target value (closing price)
- `open` - Opening price
- `high` - High price
- `low` - Low price
- `close` - Closing price
- `volume` - Trading volume

## Usage

Data files are automatically generated by:
```bash
python pull_latest_stock.py
```

## Note

CSV files are not tracked in git to keep the repository lightweight. Regenerate data as needed.
