feat: Add NewsAPI integration for news article fetching

Add comprehensive news article fetching capabilities using NewsAPI.org
to support sentiment analysis for stock market predictions.

New Features:
- fetch_news_newsapi.py: Main script to retrieve news articles from NewsAPI
- Support for AAPL, MSFT, NVDA, AMZN, GOOGL stock symbols
- Fetch articles from last 15 days (configurable up to 30 days)
- Automatic full text extraction using newspaper4k library
- Organized JSON storage in company-specific folders

API Integration:
- NewsAPI.org official REST API with 70,000+ news sources
- Environment variable or credentials file support for API key
- Built-in rate limiting (0.3s between articles, 1s between companies)
- Free tier: 100 requests/day, 100 results per request

Output Structure:
- Articles saved as JSON in outputs/{SYMBOL}/ folders
- Rich metadata: title, description, author, source, URL, image, full text
- Timestamped filenames for historical tracking

Documentation:
- NEWSAPI_README.md: Complete usage guide and API documentation
- Updated main README.md with news fetching section
- newsapi_credentials.json.template for easy setup

Dependencies:
- newspaper4k: Article content extraction
- beautifulsoup4: HTML parsing (for future enhancements)
- requests: HTTP client for API calls

This enables the stock market agent to incorporate news sentiment
analysis alongside technical indicators for improved predictions.
