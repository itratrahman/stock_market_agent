# Outputs Directory

This directory stores generated outputs including forecasts, news articles, and AI analysis reports.

## Current Contents

### Forecast Files
30-day price forecasts generated by `generate_forecasts.py`:
- `AAPL_forecast_30d_YYYYMMDD.csv` - Apple price forecast
- `AMZN_forecast_30d_YYYYMMDD.csv` - Amazon price forecast
- `GOOGL_forecast_30d_YYYYMMDD.csv` - Google price forecast
- `MSFT_forecast_30d_YYYYMMDD.csv` - Microsoft price forecast
- `NVDA_forecast_30d_YYYYMMDD.csv` - NVIDIA price forecast

**Forecast CSV Format:**
```csv
date,predicted_price,lower_bound,upper_bound
2026-01-22,249.56,237.08,262.03
2026-01-23,249.38,236.91,261.85
...
```

### AI Analysis Reports
Investment analysis reports generated by `stock_analysis_agent.py`:
- `stock_analysis_report_YYYYMMDD_HHMMSS.txt`

**Report Contents:**
- Summary table with investment decisions for all stocks
- Detailed analysis per stock:
  - Forecast summary (<300 chars)
  - News summary (<1000 chars)
  - Investment decision: INVEST / AVOID / NEUTRAL
  - Decision reasoning (<200 chars)

## Generating Outputs

```bash
# Generate forecasts
python generate_forecasts.py

# Fetch news articles
python fetch_news_newsapi.py

# Run AI analysis agent
python stock_analysis_agent.py
```
